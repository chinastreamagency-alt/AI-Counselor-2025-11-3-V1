## 🎯 重要提示

### 环境变量配置

部署时需要配置以下环境变量：

1. `NEXTAUTH_URL` - 你的 Vercel 域名
2. `NEXTAUTH_SECRET` - 从 `.env.local` 复制
3. `GOOGLE_CLIENT_ID` - Google OAuth 客户端 ID
4. `GOOGLE_CLIENT_SECRET` - Google OAuth 客户端密钥

**所有值都在 `VERCEL_ENV_READY.txt` 文件中！**

### Google Cloud Console 配置

部署后，需要在 Google Cloud Console 中添加你的 Vercel 域名：

1. 访问: https://console.cloud.google.com/apis/credentials
2. 编辑 OAuth 客户端
3. 添加 Vercel 域名到授权来源和重定向 URI
4. 保存

详细步骤在 `DEPLOYMENT_CHECKLIST.md` 中。
