## 🔄 修改密码

如需修改管理员密码，请：

1. 登录 Supabase 数据库
2. 找到 `admin_users` 表
3. 使用 bcrypt 生成新的密码哈希
4. 更新数据库记录
5. 同步更新 `.admin-config.json` 文件
