### 确保域名绑定正确

如果看到的还是旧内容，请检查：

1. **Vercel 项目绑定**
   ```
   访问: https://vercel.com/dashboard
   选择项目: AI-Counselor-2025-11-3-V1
   Settings → Domains
   确认 www.arina-ai.tech 已绑定
   ```

2. **清除浏览器缓存**
   ```
   手机浏览器：
   - Safari: 设置 → Safari → 清除历史记录
   - Chrome: 设置 → 隐私 → 清除浏览数据
   ```

3. **强制刷新**
   ```
   - iOS: 下拉刷新
   - Android: 下拉刷新
   - 桌面: Ctrl+F5 或 Cmd+Shift+R
   ```
