
## 🎨 CSS 关键对比

### object-cover vs object-contain

**object-cover（手机端）**：
```
优点：填满容器，无空白
缺点：会裁剪视频部分内容
适用：手机满屏显示
```

**object-contain（电脑端）**：
```
优点：完整显示，不裁剪
缺点：可能有黑边（如果比例不匹配）
适用：桌面完整展示
```

## 📊 技术细节

### Tailwind CSS 类解析

**h-full**:
```css
height: 100%;
/* 填满父容器高度 */
```

**aspect-video**:
```css
aspect-ratio: 16 / 9;
/* 固定宽高比 */
```

**object-cover**:
```css
object-fit: cover;
/* 裁剪视频以填满容器 */
```

**object-contain**:
```css
object-fit: contain;
/* 缩放视频以完整显示 */
```

## ⚠️ 注意事项

### 手机端 object-cover

**优点**：
- ✅ 填满屏幕，无空白
- ✅ 视觉效果好

**缺点**：
- ⚠️ 可能裁剪视频边缘
- ⚠️ 不同手机屏幕比例不同，裁剪程度不同

**建议**：
- 确保视频主要内容（人脸）在中心
- 边缘可以有一些裁剪空间

### 桌面端 object-contain

**优点**：
- ✅ 完整显示视频
- ✅ 不裁剪内容

**缺点**：
- ⚠️ 如果视频比例不是16:9，可能有黑边

**建议**：
- 确保源视频是16:9比例
- 这样就不会有黑边
